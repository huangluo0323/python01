Python基础

new方法及单例模式

我们看下这个例子

class A(object):

    def __init__(self):
        print("__init__执行了")

    def __new__(cls, *args, **kwargs):

        print("__new__ 执行了")

        return object.__new__(cls)  # 调用父类的new方法

a = A()

上面的例子我们可以看出，__new\方法调用在__init__方法之前。

讲__init__方法的时候我们讲过__init__是实例对象的初始化方法，也是就创建实例对象后调用用来初始化一些属性的方法。 __new__ 方法的调用比__init__方法还早，那么__new__作用通常用于控制生成一个新实例的过程。它是默认是一个类方法。 简单的讲，__new__方法就是用来创建实例对象的,__new__方法必须返回一个实例化对象。

初始化参数的传递问题：

class A(object):
    def __init__(self, a, b, c, k):

        print(a, b, c, k)

        print("__init__执行了")

    def __new__(cls, *args, **kwargs):

        return object.__new__(cls)   # 调用父类的new方法


a = A(1, 2365, 1122, k=3)
__new__方法创建并返回的实例对象传递给__init__方法的self参数，其他参数需要通过__new__方法传递

单例模式

单例模式是常用设计模式的一种，单例就比如我们打开电脑的回收站，在系统中只能打开一个回收站，也就 是说这个整个系统中只有一个实例，重复打开也是使用这个实例。

简单的说就是不管创建多少次对象，类返回的对象都是最初创建的，不会再新建其他对象。

单例模式写法

class SingleCase(object):
    __instance = None  # 保存实例对象
    def __init__(self,name,age):
        print(name,age)

    def __new__(cls, *args, **kwargs):

        # 如果类属性 __instance 的值为None,那么新建一个对象
        # 如果类属性值不为None 返回   __instance 保存的对象
        if not cls.__instance:
            cls.__instance = super(SingleCase,cls).__new__(cls)  # 调用父类__new__方法生成一个实例对象
            return cls.__instance
        else:
            return cls.__instance

obj1 = SingleCase('xiaoming',18)

obj2 = SingleCase('xiaoming',118)

print(id(obj1))   # id相等，说明实例化两次对象，实际上都是同一个对象
print(id(obj2))

利用类属性保存初次创建的实例对象，第二次实例化的时候判断类属性是否保存有实例对象，如果有就返回类属性保存的，如果没有就调用父类__new__方法创建新的实例对象。

注意：

单例模式中，不能调用自己的__new__方法去创建新的实例，如果调用自己会一直递归，直到报错。

错误写法：

class SingleCase(object):
    __instance = None  # 保存实例对象
    def __init__(self,name,age):
        print(name,age)

    def __new__(cls, *args, **kwargs):

        # 如果类属性 __instance 的值为None,那么新建一个对象
        # 如果类属性值不为None 返回   __instance 保存的对象
        if not cls.__instance:
            cls.__instance = cls.__new__(cls)    # 错误写法，如果调用自己的__new__方法，
                                                 # 无法创建新的实例，会递归错误
            return cls.__instance
        else:
            return cls.__instance

obj1 = SingleCase('xiaoming',18)

obj2 = SingleCase('xiaoming',118)

print(id(obj1))
print(id(obj2))

创建单例，只执行一次__init__方法

class SingleCase(object):
    __instance = None  # 保存实例对象
    __isinit = True  # 首次执行init方法标记

    def __init__(self,name,age):
        if SingleCase.__isinit:
            self.name = name
            self.age = age
            SingleCase.__isinit = False

    def __new__(cls, *args, **kwargs):

        # 如果类属性 __instance 的值为None,那么新建一个对象
        # 如果类属性值不为None 返回   __instance 保存的对象
        if not cls.__instance:
            cls.__instance = super(SingleCase, cls).__new__(cls)  # 调用父类__new__方法生成一个实例对象
            return cls.__instance
        else:
            return cls.__instance

obj1 = SingleCase('xiaoming',18)

obj2 = SingleCase('luban',118)

print(id(obj1))
print(id(obj2))
print(obj1.age)
print(obj2.age)   # 年龄都输出18。说明init值初始化了一次
